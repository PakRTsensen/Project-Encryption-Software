Inicio Rápido

Cifrado:
1. Establece una contraseña.
2. Escribe o pega texto en el área de "Texto Sin Cifrar" (borde verde).
3. Presiona el botón "Cifrar".
4. El texto cifrado aparecerá en el área de "Texto Cifrado" (borde rojo).
5. Copia el texto cifrado y úsalo donde sea necesario, o guárdalo en un archivo para uso posterior.

Descifrado:
1. Establece una contraseña.
2. Escribe o pega un texto cifrado en el área de "Texto Cifrado" (borde rojo).
3. Presiona el botón "Descifrar". (Nota: El algoritmo de descifrado se detecta automáticamente)
4. El texto descifrado aparecerá en el área de "Texto Sin Cifrar" (borde verde).
5. Copia el texto y úsalo donde sea necesario.
Esteganografía

  • Se incluye una función esteganográfica que oculta el texto dentro de una imagen (JPEG). Un algoritmo esteganográfico, el algoritmo F5, se utiliza en combinación con el algoritmo de cifrado simétrico seleccionado para crear un esteganograma final (imagen JPEG).
  • • NOTA IMPORTANTE: Algunas aplicaciones de comunicación (como WhatsApp, Viber, Messenger, etc.) pueden redimensionar o recomprimir las imágenes para reducir el ancho de banda. Los datos ocultos pueden corromperse durante este proceso porque JPG utiliza compresión con pérdida. Por favor, prueba todos tus flujos de trabajo y casos de uso previstos para verificar el proceso.

Hacer Esteganograma:
1. Establece una contraseña.
2. Escribe o pega el texto que quieres cifrar (y ocultar) en el área de "Texto Sin Cifrar" (borde verde).
3. Presiona el botón "Más" (situado a la derecha del medio, representado por tres puntos).
4. Elige "Cifrar + Hacer Esteganograma".
5. Selecciona una imagen portadora (jpg, png, bmp).
6. Presiona el botón "Continuar".
7. El esteganograma se creará y guardará.

Descifrar desde Esteganograma:
1. Establece una contraseña.
2. Presiona el botón "Más" (situado a la derecha del medio, representado por tres puntos).
3. Elige "Descifrar (Esteganograma)".
4. Selecciona un esteganograma.
5. El texto descifrado aparecerá en el área de "Texto Sin Cifrar" (borde verde).
6. Copia el texto y úsalo donde quieras.
