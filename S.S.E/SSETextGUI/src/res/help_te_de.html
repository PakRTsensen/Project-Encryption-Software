Schnellstart

Verschlüsselung:
1. Legen Sie ein Passwort fest.
2. Schreiben oder fügen Sie Text in den Bereich "Unverschlüsselter Text" ein (grün umrandet).
3. Drücken Sie die Schaltfläche "Verschlüsseln".
4. Der verschlüsselte Text erscheint im Bereich "Verschlüsselter Text" (rot umrandet).
5. Kopieren Sie den verschlüsselten Text und verwenden Sie ihn, wo immer Sie ihn benötigen, oder speichern Sie ihn für die spätere Verwendung in einer Datei.

Entschlüsselung:
1. Legen Sie ein Passwort fest.
2. Schreiben oder fügen Sie einen verschlüsselten Text in den Bereich "Verschlüsselter Text" ein (rot umrandet).
3. Drücken Sie die Schaltfläche "Entschlüsseln". (Hinweis: Der Entschlüsselungsalgorithmus wird automatisch erkannt)
4. Der entschlüsselte Text wird im Bereich "Unverschlüsselter Text" angezeigt (grün umrandet).
5. Kopieren Sie den Text und verwenden Sie ihn, wo immer Sie ihn benötigen.
Steganographie

  • Enthalten ist eine steganographische Funktion, die Text innerhalb eines Bildes (JPEG) verbirgt. Ein steganographischer Algorithmus, der F5-Algorithmus, wird in Kombination mit dem gewählten symmetrischen Verschlüsselungsalgorithmus verwendet, um ein endgültiges Steganogramm (JPEG-Bild) zu erstellen.
  • • WICHTIGER HINWEIS: Einige Kommunikationsanwendungen (wie WhatsApp, Viber, Messenger usw.) können Bilder verkleinern oder rekomprimieren, um die Bandbreite zu reduzieren. Versteckte Daten können während dieses Prozesses beschädigt werden, da JPG eine verlustbehaftete Komprimierung verwendet. Bitte testen Sie alle Ihre beabsichtigten Arbeitsabläufe und Anwendungsfälle, um den Prozess zu verifizieren.

Erstellen eines Steganogramms:
1. Legen Sie ein Passwort fest.
2. Schreiben oder fügen Sie den Text, den Sie verschlüsseln (und verstecken) möchten, in den Bereich "Unverschlüsselter Text" ein (grün umrandet).
3. Drücken Sie die Schaltfläche "Mehr" (in der Mitte rechts, dargestellt durch drei Punkte).
4. Wählen Sie "Verschlüsseln und Steganogramm erstellen".
5. Wählen Sie ein Trägerbild (jpg, png, bmp).
6. Drücken Sie die Schaltfläche "Weiter".
7. Das Steganogramm wird erstellt und gespeichert.

Entschlüsseln aus Steganogramm:
1. Legen Sie ein Passwort fest.
2. Drücken Sie die Schaltfläche "Mehr" (in der Mitte rechts, dargestellt durch drei Punkte).
3. Wählen Sie "Aus Steganogramm entschlüsseln".
4. Wählen Sie ein Steganogramm.
5. Der entschlüsselte Text erscheint im Bereich "Unverschlüsselter Text" (grün umrandet).
6. Kopieren Sie den Text und verwenden Sie ihn, wo immer Sie möchten.
